Development of a Python run tracking software¶

a presentation by Dametre Thunberg as a part of BIO-317-L¶

The goal of this project was to create tools that can be readily modified and tailored to fit an individuals need when tracking objects in movies. This includes things such as the ability to threshold objects across frame and color as well as doing kinematic calculations.¶
